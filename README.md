


## 📒 Just  Go Straight, [직진 플러그인]
> 마인크래프트 서버 플러그인

---

#### 📖 어떤 플러그인인가요?
X축 또는 Z축을 선택하여 해당 방향으로만 이동할 수 있도록 제한하는 플러그인으로 파티클을 사용하여 이동 가능 경로를 표시해줍니다.

#### 📄 명령어
```
* /직진 X - X축으로만 이동하도록 제한합니다.
* /직진 Z - Z축으로만 이동하도록 제한합니다.
* /직진 중지 - 이동 제한을 해제합니다.
```
